## はじめに
App Platform にデプロイできる Node.js を使用したサンプルアプリを提供します。これらの手順に従うことで、App Platform を使用してこのサンプルアプリケーションを実行できます。

注：以下の手順を実行すると、DigitalOceanサービスの利用料金が発生する場合があります。

## 要件
DigitalOcean アカウントが必要です。まだお持ちでない場合は、https://cloud.digitalocean.com/registrations/new でサインアップできます。
アプリのデプロイ
このボタンをクリックして、アプリを DigitalOcean App Platform にデプロイします。ログインしていない場合は、DigitalOcean アカウントでログインするよう求められます。


このボタンを使用すると、このリポジトリを直接使用しているため、リポジトリのブランチやタグにプッシュしたときにアプリを自動的に再デプロイする機能が無効になります。

アプリを自動的に再デプロイしたい場合は、GitHub リポジトリを自分のアカウントにフォークして、クラウドにコピーを保存します。GitHub リポジトリの Fork ボタンをクリックし、画面の指示に従います。

リポジトリをフォークした後、自分の GitHub 組織でこの README を表示しているはずです（例：https://github.com/<your-org>/sample-nodejs）。新しいリポジトリをデプロイするには、https://cloud.digitalocean.com/apps にアクセスし、Create App をクリックします。次に、GitHub をクリックし、作成したリポジトリを選択して main ブランチを選択します。App Platform はコードを検査し、作成するコンポーネントの種類を自動的に検出し、正しいビルドパックを使用してコンテナを作成およびデプロイします。

Deploy to DigitalOcean ボタンをクリックするか、上記の手順を完了してリポジトリをフォークした後、次の手順に従います。

HTTP ルートの指定、環境変数の設定、データベースの追加など、アプリを設定します。
アプリの名前を指定し、アプリをデプロイするリージョンを選択して、Next をクリックします。最も近いリージョンがデフォルトで選択されているはずです。すべての App Platform アプリはグローバル CDN を介してルーティングされるため、外部サービスと通信する必要がない限り、アプリのパフォーマンスには影響しません。
次の画面で、すべてのフィールドをそのままにして Next をクリックします。
Plan の設定と起動するコンテナの数を確認し、Launch Basic/Pro App をクリックします。
「Building...」という進行状況インジケーターが表示されます。View Logs をクリックすると、ビルドの詳細が表示されます。
ビルドが完了するまでに数分かかる場合がありますが、Deployments タブで進行状況を確認できます。
ビルドが正常に完了したら、ヘッダーの Live App リンクをクリックすると、新しいタブで実行中のアプリケーションが表示され、ホームページが表示されます。
アプリへの変更
リポジトリをフォークし、デプロイ時に独自のコピーを使用した場合は、フォークに変更をプッシュすると、App Platform がアプリへの更新を自動的に再デプロイするのを確認できます。これらの自動デプロイ中、App Platform はゼロダウンタイムデプロイを提供するため、アプリケーションがリクエストの処理を一時停止したり停止したりすることはありません。

このアプリで行うことができるコード変更の例を次に示します。

リポジトリ内のコードを編集します。
変更を main ブランチにコミットします。通常は、変更用に新しいブランチを作成し、レビュー後にそのブランチを main にマージするのがより良い方法ですが、このデモでは main ブランチに直接コミットできます。
https://cloud.digitalocean.com/apps にアクセスし、サンプルアプリに移動します。
最初にアプリを作成したときと同様に、「Building...」という進行状況インジケーターが表示されます。
ビルドが正常に完了したら、ヘッダーの Live App リンクをクリックすると、更新されたアプリケーションが実行されているのが表示されます。ブラウザでページを強制的に再読み込みする必要がある場合があります（例：Shift+Reload を使用）。
詳細
App Platform の詳細と、アプリケーションの管理および更新方法については、https://www.digitalocean.com/docs/app-platform/ を参照してください。

## アプリの削除
このサンプルアプリケーションをライブで実行する必要がなくなったら、次の手順に従って削除できます。

https://cloud.digitalocean.com/apps の Apps コントロールパネルにアクセスします。
サンプルアプリに移動します。
Settings タブで、Destroy をクリックします。
注：アプリを削除しないと、DigitalOcean サービスの利用料金が引き続き発生します。
